Biên code aka lậptrành, với trợgiúp cụa các IDE, chúng cô chỉ cần nhớ các kháiniệm sau:

1. Phép gán

Đây là thaotác đưa giátrị vào một thamsố, chảdụ X=1 nghĩa là gán cho thamsố (parameter/variable) X giátrị 1.

Còn X=X+1 nghĩa là làm X tăng lên 1 đơnvị, thường zùng cho số nguyên.

2. Phép so

Đây là thaotác sosánh một thamsố chotrước với một đốitượng nhấtđịnh, gòi tùy theo kếtquả, mà rẽ-nhánh chươngtrình thựchiện.

Chảdụ như này:

IF x>0 THEN
PRINT "X lớn hơn zero"
ELSE
PRINT "X bé hơn zero"

Đoạn code đó sosánh x với 0, lếu x>0 thì in "X lớn hơn zero" ra mànhình, ngượclại thì in "X bé hơn zero"

3. Phép lặp

Đây là một đoạn code nhằm thựchiện một việc gì mà lặp-đi-lặp-lại. Nó gồm có từkhóa mở vònglặp, và đóng vònglặp.

Chảdụ

x=1
WHILE x<10
PRINT x
x=x+1
END

Ở đây, WHILE là từkhóa mở vònglặp, còn END là từkhóa đóng vònglặp đó (tránh chươngtrình bị treo mẹ zo cứ hoạtđộng mãi mà đéo kếtthúc).

Đoạn code trên sẽ in các consố từ 1 đến 9 ra mànhình.

Thoạtđầu, nó gán cho thamsố (biếnsố) x giátrị 1, rồi trong vònglặp, nó in x ra mànhình, rồi tăng x lên 1 đơnvị, rồi sosánh với 10, nếu x nhỏ hơn 10, thì in x ra, còn ngượclại, thì thoát khỏi vònglặp.
