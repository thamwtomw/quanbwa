RAO-MUA KÈN BỰA BẰNG WEB3 JS BỰA CONSOLE: DÙNG VÍ GIẤY CÁNHÂN

Nhândân sẽ.

1. vào Quán Bựa qua any Web Browser;
2. bật Web JS Console, bằng cách bấm phím F12;
3. gọi tươngtác Gemt9 or Nemt9 trong JS Console;
4. import khóa riêng của ví giấy cánhân trong JS Console;
5. chạy lệnh rao-mua kèn, Gemt9 or Nemt9, trong JS Console;
6. khi rao-mua thànhcông, chạy lệnh xóa History của JS Console;

Bước (3) thựchiện như khi mua/bán kèn.

 startGemtRpc(); 

cho kèn Gemt9, hoặc

 startNemtRpc(); 

cho kèn Nemt9.

Bước (4) cũng thựchiện như khi mua/bán kèn.

 importWallet(WALLET_PRIVATE_KEY); 

Bước (5) thựchiện bằng cách chạy lệnh này:

 marketEtherSellRaw(ORDER_ID,COIN_AMOUNT,BID_PRICE); 

ORDER_ID là sốhiệu Order/Bid ID nhândân chọn.

COIN_AMOUNT là sốlượng BNB native-coin nhândân tính dùng mua Kèn Bựa.

BID_PRICE, hay TOKENS_PER_COIN, là giá kèn nhândân rao-mua.

Chảdụ, sốhiệu Order/Bid ID chọn bằng "654321", sốlượng rao-mua 15,000 Gemt9, giá rao-mua 20,000 BNB/GEMT9, aka 1 BNB ăn 20,000 Gemt9.

15000/20000 = 0.75 BNB (chưa gồm phí giaodịch, quãng 0.0002 BNB)

Lệnh trên sẽ thành:

marketEtherSellRaw (654321, 0.75, 20000);

hoặc

marketEtherSellRaw ("654321", "0.75", "20000");

 clearHistory();
